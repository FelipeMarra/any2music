# T5
T5 is the text encoder used in [MusicGen](https://musicgen.com/). It was described in the paper [Exploring the Limits of Transfer Learning with a Unified Text-to-Text Transformer](https://arxiv.org/abs/1910.10683).

### Example Usage
Please reffer to the test file at [tests/text/encoders/test_t5.py](../../../tests/text/encoders/test_t5.py) for usage example.
